# 🫧 BadBleep Box — private cheat registry

> Internal doc. Deliberately a dot-file, not linked from README or the app.
> The shipped bundle only contains **hashes** of these codes
> (`src/engine/bleepbox.js`), so reading the deployed JS won't reveal them.

## Where

**You tab → ⚙ Settings → scroll to the bottom → "🫧 BadBleep Box"** — type a
code and hit *bleep* (or Enter). Codes are case/space/punctuation-insensitive.

## Adding a new code

1. Compute the hash:
   ```bash
   node -e "const n=s=>s.toLowerCase().replace(/[^a-z0-9]/g,'');const h=s=>{let x=0;for(const c of n(s))x=(x*31+c.charCodeAt(0))>>>0;return x.toString(36)};console.log(h('YourNewCode'))"
   ```
2. Add `"<hash>": { fx: "<effect>" }` to `CODES` in `src/engine/bleepbox.js`.
3. Implement the `fx` branch in `onCheat` in `src/App.jsx`.
4. Record it in the table above. Do not write the plaintext anywhere else.

## Notes

- Effects are **local-device only** (they mutate the local save blob); nothing
  is granted server-side, so codes can't inflate the party feed or other users.
- `SupaBleepMode` is additive — it won't remove anything you already own, and
  the state persists via the normal save. To reset: Settings → Clear all local data.
- The 31-rolling hash is anti-casual, not cryptographic. Fine for fun codes;
  don't gate anything valuable behind it.
